# Device

# Properties

  • name String (opens new window)? Device type, eg. Mobile
  • width String (opens new window)? Width to set for the editor iframe, eg. '900px'
  • height String (opens new window)? Height to set for the editor iframe, eg. '600px'
  • widthMedia String (opens new window)? The width which will be used in media queries, If empty the width will be used
  • priority Number (opens new window)? Setup the order of media queries
